# Daehyeok Kim GitHub Pages

## Deploy
1. Create a repository named `daehyeok48.github.io`.
2. Upload all files in this folder to the repository root.
3. Open **Settings → Pages**.
4. Select **Deploy from a branch**, then `main` and `/root`.
5. Visit `https://daehyeok48.github.io`.

## Images
Replace the gray placeholder blocks in `index.html`, `project.html`, and other pages with:
```html
<img src="assets/your-image.jpg" alt="Description">
```
Then add this CSS if needed:
```css
.card-image img { width:100%; height:100%; object-fit:cover; }
```
